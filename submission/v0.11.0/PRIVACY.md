# 隐私政策

最后更新：2026-07-12

“期刊会议等级与分区助手”用于在 Google Scholar、DBLP、Semantic Scholar、arXiv、OpenAlex、PubMed、知网和万方 等学术检索页面旁显示期刊与会议等级信息。

## 数据处理

本扩展不收集、出售或向开发者服务器上传用户数据，不包含广告、统计分析、追踪器或用户账户系统。

扩展会在浏览器本地读取当前学术检索页面中的论文标题、期刊/会议名称、ISSN 和检索词，用于匹配等级并显示标签。这些数据默认只在本机处理。

以下功能会产生必要的 HTTPS 网络请求：

- 从本项目 GitHub 仓库检查并下载签名数据库更新。请求不包含当前浏览页面或检索内容。
- DBLP 页面服务不可用时，将当前 DBLP 检索词发送到 DBLP 官方 API，以恢复论文结果。
- 期刊缺少发行商或研究方向时，将期刊名称或 ISSN 发送到 Crossref API。结果在浏览器本地缓存 30 天。

开发者不会接收或保存上述请求内容。相关第三方服务对其收到的数据适用各自的隐私政策。

## 本地存储

扩展在浏览器本地保存：

- 用户选择的显示选项和主题；
- 加密的期刊/会议等级数据库；
- 数据库更新状态；
- Crossref 期刊元数据缓存。

卸载扩展会由浏览器删除这些本地数据。扩展不使用云同步存储。

## 权限用途

- `storage`：保存设置、加密数据库和缓存。
- `unlimitedStorage`：容纳完整的本地等级数据库，避免浏览器存储配额导致更新失败。
- `alarms`：每 7 天检查一次数据库更新。
- 学术网站访问权限：识别页面上的论文来源并显示等级标签。
- `raw.githubusercontent.com`：下载本项目维护者签名的数据包。
- `api.crossref.org`：按需补全期刊发行商和研究方向。

## 数据安全

所有网络请求均使用 HTTPS。数据库更新必须通过 ECDSA P-256 签名和 SHA-256 完整性验证；本地数据库分片使用 AES-GCM 加密。
